Student Management System (Django)

A simple and clean **Student Management System** built using **Django**.  
This project allows you to **Add, Edit, Update, and Delete** student records easily.


# Features

- Add new students  
- Update student information  
- Delete student records  
- View all students in a table  
- Form validation
